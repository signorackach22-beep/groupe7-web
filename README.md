# Mini User Manager

TP 1 — React.js : Composants, State et Appels API  
Cours : L2-S4-DAWM — Outils de développement web  
Enseignant : Dr Abakar Issakha Souleymane  
École : ENASTIC — Année académique 2025-2026

## Lancer le projet

```bash
npm install
npm run dev
```

## Fonctionnalités

- Affichage des utilisateurs depuis une API externe
- Ajout / Suppression d'utilisateurs
- Recherche dynamique en temps réel
- Compteur avec limite maximale
- Interface responsive
